# AI Comparison Tool
=======================

## Description
The AI Comparison Tool is a Python-based application designed to compare the performance of various AI providers. The tool allows users to configure multiple providers, test them with custom prompts, and analyze the results.

## Installation
To install the AI Comparison Tool, follow these steps:

1. Clone the repository using `git clone`.
2. Install the required dependencies using `pip install -r requirements.txt`.
3. Configure your API keys for each provider in the `ai_comparison.py` file.

## Usage
To use the AI Comparison Tool, simply run the `ai_comparison.py` file using `python ai_comparison.py`. The tool will guide you through the setup and comparison process.

## Example
Here's an example of how to use the AI Comparison Tool:

1. Run the tool using `python ai_comparison.py`.
2. Configure the providers you want to compare by selecting the "Setup providers" option.
3. Test the providers with a custom prompt by selecting the "Single prompt comparison" option.
4. Analyze the results and view the performance trends for each provider.
