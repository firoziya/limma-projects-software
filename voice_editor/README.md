**Voice-Controlled Text Editor**
================================

**Description**
---------------

A simple voice-controlled text editor built using Python. This editor allows users to create, edit, and manage text files using voice commands.

**Installation**
---------------

To install the required libraries, run the following command:
```bash
pip install limma keyboard
```
**Usage**
-----

1. Run the editor using `python voice_editor.py`.
2. Follow the on-screen instructions to use the voice-controlled editor.
3. Say a command or start dictation to interact with the editor.

**Example**
-------

Some example voice commands:

* "New" - Create a new file
* "Open" - Open an existing file
* "Save" - Save the current file
* "Save as" - Save the current file with a new name
* "Read" - Read the current content aloud
* "Clear" - Clear all content
* "Add" - Add text to the content
* "Delete" - Delete text from the content
* "Find" - Find text in the content
* "Replace" - Replace text in the content
* "Word count" - Count the words and characters in the content
* "Time" - Insert the current time
* "Date" - Insert the current date
* "Help" - Show available voice commands
