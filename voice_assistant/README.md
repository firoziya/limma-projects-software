# Smart Voice Assistant
================================

## Project Description
Smart Voice Assistant is a Python-based project that utilizes natural language processing (NLP) and machine learning (ML) to create a voice-controlled assistant. The assistant can perform various tasks, such as telling the time and date, searching the web, taking notes, and even telling jokes.

## Installation
To install the Smart Voice Assistant, follow these steps:

1. Clone the repository using `git clone https://github.com/firoziya/smart-voice-assistant.git`
2. Install the required libraries by running `pip install -r requirements.txt`
3. Set up your Gemini API key by exporting the `GEMINI_API_KEY` environment variable
4. Run the assistant using `python voice_assistant.py`

## Usage
To use the Smart Voice Assistant, simply run the `voice_assistant.py` script and follow the prompts. You can customize the voice settings by selecting the "Customize voice settings" option when prompted.

## Example
Here's an example of how to use the Smart Voice Assistant:

* Say "What's the time?" to hear the current time
* Say "Search for Python tutorials" to search for Python tutorials on Google
* Say "Take a note" to save a note
* Say "Tell me a joke" to hear a joke
