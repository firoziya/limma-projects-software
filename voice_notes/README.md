# Voice Notes App
A voice-activated note-taking application built with Python.

## Description
This application allows users to create, search, read, edit, delete, and categorize notes using voice commands. It also provides a summary of notes and allows users to set reminders.

## Installation
To install the required libraries, run the following command:
```bash
pip install limma
```
Make sure to install the `limma` library, which is used for voice recognition and synthesis.

## Usage
To run the application, simply execute the `voice_notes.py` file:
```bash
python voice_notes.py
```
The application will start listening for voice commands. Available commands include:

* `create note`: Create a new note
* `search notes`: Search for notes
* `read notes`: Read notes aloud
* `edit note`: Edit an existing note
* `delete note`: Delete a note
* `set reminder`: Set a reminder for a note
* `categorize`: Categorize or re-categorize notes
* `summary`: Get a summary of notes
* `exit`: Quit the application

## Example
Here's an example of how to use the application:
1. Run the application using `python voice_notes.py`.
2. Say "create note" to create a new note.
3. Speak the content of the note when prompted.
4. Select a category for the note (e.g., "Personal", "Work", etc.).
5. Say "read notes" to listen to your notes.
6. Say "edit note" to edit an existing note.
7. Say "delete note" to delete a note.
8. Say "set reminder" to set a reminder for a note.
9. Say "categorize" to categorize or re-categorize notes.
10. Say "summary" to get a summary of your notes.
