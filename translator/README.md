# AI-Powered Language Translator
==============================

## Description
This is a Python-based language translator project that utilizes AI-powered technology to translate text and speech in real-time. The project supports multiple languages and features various modes, including text translation, voice translation, conversation mode, and batch translation.

## Installation
To install the required dependencies, run the following command:
```bash
pip install -r requirements.txt
```
Make sure to replace `requirements.txt` with the actual file containing the dependencies required by the project.

## Usage
1. Clone the repository using `git clone https://github.com/firoziya/ai-translator.git`
2. Navigate to the project directory using `cd ai-translator`
3. Run the project using `python translator.py`
4. Follow the on-screen instructions to select the desired translation mode and language.

## Example
To translate a sentence from English to Spanish, follow these steps:
1. Run the project using `python translator.py`
2. Select option 1 (Text Translation) from the main menu
3. Enter the text to translate (e.g., "Hello, how are you?")
4. Select the target language (e.g., Spanish)
5. The translated text will be displayed on the screen
