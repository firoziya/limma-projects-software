# Story Generator Project
=========================

## Project Title
Interactive Story Generator

## Description
This project is an interactive story generator that uses AI to help users create and develop their own stories. It provides features such as generating random story ideas, creating new stories, developing characters, generating plot twists, and voice storytelling.

## Installation
To install this project, follow these steps:

1. Clone the repository using `git clone https://github.com/firoziya/story-generator.git`
2. Install the required libraries using `pip install -r requirements.txt`
3. Set up your environment variables, including `GEMINI_API_KEY` for the AI model

## Usage
To use the story generator, follow these steps:

1. Run the project using `python story_generator.py`
2. Follow the prompts to start a new story, load a saved story, or generate a random story idea
3. Use the available commands to develop your story, including `new`, `load`, `chapter`, `character`, `twist`, `tell`, `save`, and `list`
